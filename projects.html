Development of an Aﬀordable Open-Sky Planetarium (Ongoing)

This project involves the development of an open-sky planetarium using stepper motors in a pan-tilt mechanism. The device is being controlled by a plug-in running on a computer, which sends the number of steps required for a movement. Accordingly, a LASER mounted at the shaft of tilt motor points to the target star.
Project supervisor: Prof. Kannan M. Moudgalya (Professor, IIT Bombay)

Tasks involved:
  • Developing hardware architecture for the project.
  • Interfacing hardware components with a GUI.
  • Improving the methodology to calculate coordinates for high precision.
  • Developed a sleep mechanism for microcontroller to save power.
Technical tools used: Scilab, Arduino IDE, Qt, KiCad

Components used: Stepper motors, Arduino Nano, Relay, Transistor.

Development of a Landslide Risk Communication System (December 2016 - April 2017)

This project involved the development of a low-cost early warning system for landslides using Arduino board and low-cost sensors. A solar-powered weather station was developed, which was responsible for measuring weather conditions and soil parameters. Once these values crossed a threshold, an alert was sent to registered users on their phones via a GSM module.
Project supervisor: Dr. Varun Dutt (Assistant Professor, IIT Mandi)

Tasks achieved:
  • Developed a solar-powered weather station.
  • Interfaced sensors with microcontroller and GSM module.
  • Designed the IoT framework of the weather station.
  • Developed a sleep mechanism for microcontroller to save power.
Technical tools used: Arduino IDE, PLX-DAQ Excel

Components used: Arduino Mega, Soil-moisture sensor, Accelerometer, Tilt sensor, GSM module, Solar Panel, Li-ion rechargeable batteries.

Localization of Wireless Charging Vehicles in a network

In this project, I studied the utility of wireless charging vehicles (WCV) in a wireless sensor network. In order to maximize the lifetime of a network, this study proposed the stationing of more than one WCV in a single network. With this, a need of collaboration among the WCVs was found out.
Project supervisor: Dr. Ronald Y. Chang (Research Fellow, Academia Sinica)

Tasks achieved:
  • Examined the utility of WCV for powering nodes in a wireless sensor network.
  • An optimum path was proposed for WCV to ensure the life of nodes.
  • High energy requirements of nodes close to base station were ascertained to address energy hole issue.
Technical tools used: MATLAB

Development of Scilab-Arduino Toolbox

In this project, we worked for the testing of Scilab-Arduino ToolBox (open source). Thus, we developed codes in Scilab to program an Arduino board to run components like LDC Motors, Push Buttons, Thermistors, etc. Accordingly, a Scilab-Arduino workshop was conducted by FOSSEE, where the participants were given a hands-on using these codes.
Project supervisor: Prof. Kannan M. Moudgalya (Professor, IIT Bombay)

Tasks achieved:
  • Interfaced components with Arduino board.
  • Prepared documentation for the steps involved.
  • Tested codes for Linux and Windows.
Technical tools used: Arduino IDE, Scilab

Components used: Arduino UNO, LEDs, LDRs, DC Motors, Push Buttons, Thermistors and Servo Motors.
